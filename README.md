# Tourism AI Advisor

A modern static chatbot-style landing page for a travel assistant. The project uses plain HTML, CSS, and JavaScript to simulate a travel advisor interface that responds to user prompts and seasonal tour queries.

## Features

- Responsive travel advisor landing page
- Animated UI with floating icons and card interactions
- Chat-style conversation area with user and bot messages
- Predefined suggestion cards for quick actions
- Simple rule-based bot responses for greetings, tour recommendations, prices, and travel booking guidance

## Project Structure

- `index.html` - page markup and UI layout
- `style.css` - styling, animations, and responsive design
- `script.js` - chat behavior, user input handling, and bot response logic

## Usage

1. Open `index.html` in your browser.
2. Click **Start Chat** or use one of the predefined message cards.
3. Type a question in the input field and press **Enter** or click the send button.
4. The chat assistant will reply with a travel-related response.

## Notes

- This is a static front-end demo and does not connect to any backend or real AI service.
- The bot logic is implemented with simple keyword matching inside `script.js`.

## Customize

- Update `tourData` in `script.js` to change seasonal tour pricing and details.
- Modify `style.css` to adjust the visual theme, animations, and layout.
- Add more predefined cards in `index.html` to provide additional quick prompts.
